# GitHub Viewer
Neste projeto foi utilizado AngularJs para recuperar alguns dados da API do GitHub.
 
## Características
* Busca de usuários
* Listagem de repositórios
* Filtragem de repositórios
* Listagem de Issues
 
## Ferramentas utilizadas
* AngularJs
* Bootstrap
* SASS
* Grunt
* Bower
* Yeoman

## Como instalar

Instalar as dependências

```
$ npm install
```


Rodar o projeto

```
$ grunt serve
```
