Technologies
==============
Angular
ES6
Babel
Karma
Jasmine
Bootstrap
SCSS
Webpack

Setup
==============
npm install

Run server
==============
npm run dev

Run test
==============
npm run test

Features
==============
- List repositories by stargazers_count order from elastic search github api
- Get stars and forks of repo
- Get commits from repo
- Load more commits of repo, if less than 20 hide load more
